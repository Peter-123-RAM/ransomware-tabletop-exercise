# Ransomware Tabletop Exercise

## Overview

This project contains a complete 90-minute ransomware tabletop exercise designed for the executive leadership team of a 50-person African retail company.

The exercise simulates a modern double-extortion ransomware attack involving operational disruption, customer data theft, regulatory pressure, media scrutiny, and executive decision-making.

The package includes:

- Scenario overview
- Four timed scenario injections
- Executive role briefs
- Facilitator guide
- Decision tracking templates
- After-action review template
- Printable PDF version

## Objectives

1. Evaluate executive incident response readiness.
2. Assess business continuity decision-making.
3. Test communication and legal response processes.
4. Identify gaps in ransomware preparedness.
5. Produce actionable lessons learned.

## Audience

- CEO
- COO
- CFO
- General Counsel
- Head of Communications

## Duration

90 Minutes

## Scenario Theme

Double-extortion ransomware attack against a regional retail company operating physical stores and online ordering systems.
